This is a simple game to be played to introduce the students to printing on the LED screen. The user enters a codeword into the program in the beginnging, and then uploads the code to the board. He/she then asks a player to guess a letter. If the player guesses the correct letter, the user pressers the "correct" button and a green LED lights up. If the player is wrong, the user presses the "incorrect" button and the red LED illuminates. Regardless of right or wrong, after the push of a button, the next letter appears on the screen. THis process is repeasted until the entire word is displayed on the screen. The screen then tells the player whether they guesses the majority of the letters correct or incorrect
